# Path C: Hybrid 90-Day Plan

Objective: Execute growth as primary while preserving credible exit optionality.

## Operating Split

1. 80% effort on growth and moat execution.
2. 20% effort on diligence readiness and strategic relationships.

## Days 1-30

1. Define explicit triggers for Exit vs Empire decision at 24 months.
2. Build dual-track KPI dashboard.
3. Assign one owner for growth track and one for readiness track.
4. Publish internal communication to avoid ambiguity.
5. Confirm board agreement on hybrid governance.

## Days 31-60

1. Launch priority growth initiatives with measurable outcomes.
2. Keep financial and legal readiness current.
3. Maintain selective relationship mapping with strategic buyers.
4. Run monthly decision-risk review.
5. Enforce strict focus to prevent diluted execution.

## Days 61-90

1. Evaluate track performance using predefined triggers.
2. Remove low-leverage initiatives.
3. Rebalance effort if one track underperforms.
4. Publish checkpoint memo to board.
5. Confirm next 90-day hybrid plan or pivot to A/B.

## Key Metrics

1. Growth-track KPI attainment
2. Readiness-track completeness score
3. Leadership clarity and team confidence signal
4. Strategic optionality quality
5. Decision trigger trajectory
