# Path B: Empire 90-Day Plan

Objective: Commit to long-horizon scale with stronger leadership depth, product leverage, and moat execution.

## Days 1-30

1. Run leadership offsite and lock 3-year strategic thesis.
2. Define top 3 growth bets and sequence by ROI and risk.
3. Finalize executive hiring priorities and owners.
4. Build updated operating model and capital plan.
5. Communicate long-term direction to team.

## Days 31-60

1. Launch first growth bet and publish KPI baseline.
2. Start leadership hiring pipeline for critical roles.
3. Prioritize product moat roadmap items.
4. Build adjacent-market validation experiments.
5. Implement monthly strategic scorecards.

## Days 61-90

1. Review early signal quality from growth bets.
2. Tighten resource allocation to top-performing tracks.
3. Finalize 12-month hiring and execution calendar.
4. Establish quarterly board strategy cadence.
5. Lock Year 1 milestones for empire build.

## Key Metrics

1. ARR growth quality
2. Net revenue retention
3. Executive hiring progress
4. Moat feature adoption
5. Adjacent-market validation score
