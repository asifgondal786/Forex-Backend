# Phase 12 Week 1 Actions

## Universal Actions

1. Run leadership decision workshop and complete `PHASE_12_DECISION_SCORECARD.md`.
2. Prepare and circulate a board memo using `BOARD_PATH_DECISION_MEMO_TEMPLATE.md`.
3. Draft current-state snapshot using `STATE_OF_COMPANY_TEMPLATE.md`.
4. Build 90-day milestones using the selected path plan.
5. Initialize `PHASE_12_DECISION_LOG.csv` and `PHASE_12_90_DAY_TRACKER.csv`.

## Deliverables By End Of Week

1. Path selected and approved.
2. Owner assigned for every milestone in first 30 days.
3. Top 5 strategic risks and mitigations documented.
4. Weekly cadence meetings scheduled for next 12 weeks.
5. Board and leadership communication sent.

## Non-Negotiables

1. No silent strategy pivots outside decision log.
2. No unowned milestones.
3. No major commitment without board visibility.
