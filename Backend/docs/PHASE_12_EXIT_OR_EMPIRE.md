## Decision Rules

1. Pick one path and commit for 90 days.
2. Do not run mixed priorities unless Path C is explicitly approved.
3. Re-evaluate only at scheduled checkpoints.
4. Tie decisions to metrics, not narrative.

## Core Metrics

1. Revenue growth quality (ARR, margin, retention)
2. Strategic leverage (buyer interest or moat depth)
3. Team readiness (leadership depth, execution velocity)
4. Risk profile (concentration, legal, technical, people risk)
5. Founder energy and time horizon

## Governance Cadence

1. Weekly: leadership execution review using 90-day tracker.
2. Monthly: board update with scorecard deltas and blockers.
3. Day 30 / 60 / 90: path checkpoint with explicit continue/adjust call.

## Exit Criteria For Phase 12

1. Decision rationale documented and approved by leadership.
2. One path is actively executed with measurable milestones.
3. A decision log exists for major strategic changes.
4. 90-day tracker shows on-time execution.
